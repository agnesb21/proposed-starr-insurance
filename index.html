📌 Design Notes for Dev Handoff

Nudge Banner Appears only when user arrives via the "View & Compare Plans" CTA (pass a URL param e.g. ?ref=cta-compare). Dismissable. Sticky to top of comparison section on scroll.
Soft Lock Overlay Shown when form is NOT yet filled. Table rows are visible but "Select Plan" buttons are disabled. Clicking overlay CTA smooth-scrolls user back up to the quote form.
Unlock Trigger Once user submits the quote form, overlay disappears, real pricing populates, and "Select Plan" buttons activate. No page reload needed — JS state toggle.
